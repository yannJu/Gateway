## Rpi 에서 Django 실행하기🏴‍☠️

1. ### [라즈베리파이를 서버로 만들기(mjpeg/)](./mjpeg/)
   - 외부 **브라우저(웹)**에서는 **라즈베리파이** 서버에 직접 연결 불가능
   - 따라서 **라즈베리파이** 에서 `django` 서버를 연동
   - 예를들어 **실시간 스트리밍** 을 하고싶을때 사용 가능
   - `MJpeg` 용 **HTTP** 응답 메시지를 직접 만들어야 한다.
      - `<img src = "url~~">` 을 이용하여 **mjpeg** 요청이 오면 그에 대한 응답으로 `응답코드` > `헤더` > `바디` . . 로 이루어진다.
      - `바디` 내에 여러 **part** 들이 있는데, 각 **part** 도 `헤더` > `빈줄` > `바디` 의 구조를 가진다 . . 
   - 응답 메시지를 `View` 에서 처리할 수 있어야 한다.
2. ### [라즈베리파이에서 읽어온 데이터를 웹브라우저 서버로 띄우기(/mjpeg/cam.py)](./mjpeg/cam.py)
   - **이미지 캡쳐** 혹은 **동영상 녹화** 기능을 구현하기 위해 `MJpegStreamCam` 클래스 생성
   - `Byte` 로 진행할 예정이므로 **frame** > **jpg** > **byte**  순으로 변환
   - `__iter__` 함수에서 **yield** 를 이용해 제너레이터 구현
      - **yield** 를 이용하면 보다 빠르게 반복해서 **반환** 할 수 있다.
   - *[mjpeg/views.py](./mjpeg/views.py)* 에 이미지 캡쳐를 위한 `snapshot` 함수와 동영상 녹화를 위한 `stream` 함수를 통해 **view** 생성
