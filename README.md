	- `GPIO` 를 이용하여 **입출력** 연습

    - **아두이노** 에서 사용한 센서들을 이용하여 간단한 실습 진행
2. ### [GPIO_ZERO(ex11_ex)](./GPIO_ZERO/)
    - `gpiozero` 라이브러리를 이용하여 보다 센서를 쉽게 다룰 수 있다.
